## 项目介绍
基于ssm的地球物理台管理系统：前端 jsp、jquery、bootstrap，后端 maven、springmvc、spring、mybatis；集成站台管理、相关学科管理等功能于一体的系统。

## 功能介绍

- 基本功能：登录，退出，密码修改
- 站台列表：站台列表查询，按名称和学科搜索查询，添加站台，编辑和删除
- 其他功能：添加学科（强震、地磁、流体、GNSS、重力、形变、辅助）
